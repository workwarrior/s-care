# S-Care

Smart Care Inspection Platform

Offline First Inspection Platform untuk Rumah Sakit, Klinik, dan Perusahaan.

## Fitur

- ✅ Offline First
- ✅ Progressive Web App
- ✅ Android App (Capacitor)
- ✅ Google Apps Script Backend
- ✅ GitHub Pages Hosting

## Teknologi

- Vite
- Vanilla JavaScript
- Bootstrap 5
- IndexedDB
- Capacitor
- Google Apps Script

## Menjalankan Project

```bash
npm install
npm run dev
```
