# S-Care Architecture

## Tujuan

S-Care adalah platform inspeksi berbasis Web dan Android dengan pendekatan Offline First.

## Arsitektur

Frontend
↓
Router
↓
Feature
↓
Repository
↓
IndexedDB
↓
Sync Engine
↓
Google Apps Script

## Teknologi

- Vite
- Vanilla JavaScript
- Bootstrap 5
- Bootstrap Icons
- IndexedDB
- Capacitor (Android)
- Google Apps Script
- GitHub Pages

## Prinsip

- Offline First
- Mobile First
- Modular
- Single Page Application (SPA)
